# AGENTS.md

## Project
**LifeOS** — Build a premium AI life operating system web app first, then later expand to iOS/Android.

## Primary Goal
Deliver a production-ready, scalable MVP with a premium user experience and strong daily-retention workflows.

## Tech Stack
- Next.js
- React
- TailwindCSS
- Framer Motion
- shadcn/ui
- Supabase
- Stripe
- OpenAI

## Build Rules
1. Build **mobile-first** with responsive layouts.
2. Use a **luxury dark fintech** visual style.
3. Keep components modular and reusable.
4. Do not overbuild the MVP; prioritize high-impact scope.
5. Prioritize core modules: dashboard, AI assistant, tasks/goals, calendar, finance hub, automation, vault.
6. Avoid changing unrelated files.
7. Never expose API keys or secrets.
8. Add clear user-facing and server-side error handling.
9. Keep code production-ready (typed, tested where practical, maintainable).
10. Before final response, summarize changed files and recommended next steps.
